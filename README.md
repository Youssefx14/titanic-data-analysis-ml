Titanic Survival Analysis & Prediction

Project Overview

This project analyzes the Titanic dataset to understand the factors that influenced passenger survival and builds a machine learning model to predict survival outcomes.
The goal is to practice data analysis, visualization, and basic machine learning using Python.

________________________________________________________________________________________________________________________________
Dataset

>Source: Kaggle – Titanic Competition

>Link: https://www.kaggle.com/competitions/titanic/data

Key Features

>Survived – Survival (0 = No, 1 = Yes)

>Pclass – Passenger class

>Sex – Gender

>Age – Age in years

>SibSp – Number of siblings / spouses aboard

>Parch – Number of parents / children aboard

>Fare – Ticket fare

>Embarked – Port of embarkation
